## About
This is my second (updated) portfolio website built using **React-Bootstrap**

## Technical Stack

* **Framework:** React.js
* **Styling:** React-Bootstrap
* **Deployment:** Github Pages
* **Forms:** Web3Forms integration

## Commands
This project uses npm scripts to manage the build and deployment process.

| Command | Description |
| :--- | :--- |
| `npm install` | Installs all necessary project dependencies. |
| `npm run build` | Compiles the source code into optimized production files. |
| `npm run deploy` | Pushes the build directory to the production environment. |

To run the project locally during development:
```bash
npm run dev
